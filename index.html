Nyzo tech

This is the site for Nyzo technical documentation. This supersedes the white paper as a technical description of Nyzo. The white paper, available on the web site, should be considered a historical document.

This was made in the form of a public git repository to allow for transparent, low-effort updating.

The content of this site is available at https://github.com/n-y-z-o/tech. The code that runs the site is available at https://github.com/n-y-z-o/nyzoVerifier. Instructions for running a mirror or variant of this site are also available.
